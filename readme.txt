ScriptData Class :

The ScriptData class provides methods to fetch and convert intraday data for a given stock ticker. The class uses the Alpha Vantage API for data retrieval and Pandas for data manipulation.

Methods

fetch_intraday_data(stk): This method fetches intraday data for the given stock ticker stk and stores the data in the map attribute of the class.

convert_intraday_data(stk): This method converts the raw intraday data stored in the map attribute for the given stock ticker stk to a Pandas DataFrame with appropriate data types and column names. The converted data is also stored in the map attribute and is returned by the method.

__init__(): This method initializes the map attribute to an empty dictionary and sets the Alpha Vantage API key to a default value.

__getitem__(stk): This method allows access to the intraday data for the given stock ticker stk using dictionary-like square bracket notation.

__setitem__(key, value): This method allows setting the intraday data for a given stock ticker key using dictionary-like square bracket notation.

__contains__(key): This method allows checking if the intraday data for a given stock ticker key is present in the map attribute using dictionary-like square bracket notation.

indicator1 function :

This function calculates a simple moving average of closing prices for a given stock using a specified time period, and returns a dataframe containing the resulting indicator values.

Inputs
df (Pandas DataFrame): A dataframe containing intraday data for a given stock, including columns for 'timestamp' and 'close'.
timeperiod (int): The time period (in minutes) over which to calculate the moving average. Defaults to 5 minutes.
Outputs
indiframe (Pandas DataFrame): A dataframe containing the calculated moving average values and their corresponding timestamps.

The Strategy class is designed to perform technical analysis on stock data. The __init__ method initializes the class by fetching intraday stock data and storing the closing price and technical indicator data. The get_script_data method returns the stored closing price and indicator data. The get_signal method analyzes the data and generates trading signals, which are stored in a DataFrame named signal. The plot_action method plots the closing price and indicator data, and displays markers to indicate the buy and sell signals generated by the get_signal method. To use this class, an instance must be created by providing a stock symbol to analyze, then the get_signal and plot_action methods must be called on the instance.
